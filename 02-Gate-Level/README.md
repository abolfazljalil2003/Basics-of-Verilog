# سطح گیت (Gate Level)

در این بخش با استفاده از گیت‌های آماده Verilog مدارهای ترکیبی می‌سازیم.

## محتوای این بخش

### گیت‌های پایه
- AND, OR, NOT, NAND, NOR, XOR, XNOR

### مدارهای ترکیبی ساده
- Half Adder
- Full Adder
- Comparator

### مدارهای پیچیده‌تر
- Ripple Adder
- Multiplexer
- Demultiplexer
