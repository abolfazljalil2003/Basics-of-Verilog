# آموزش Verilog به زبان فارسی

یک منبع برای یادگیری طراحی سخت‌افزار دیجیتال با Verilog

## درباره این پروژه

این مخزن حاصل یادگیری یک ترم درس برنامه‌نویسی Verilog است و شامل مجموعه‌ای کامل از آموزش‌ها، مثال‌ها و تست‌بنچ‌های عملی می‌باشد. تمام مطالب به زبان فارسی و با رویکرد عملی نوشته شده‌اند.

## ویژگی‌های این آموزش

- آموزش از صفر تا صد به زبان فارسی
- مثال‌های عملی برای هر مفهوم
- تست‌بنچ کامل برای تمام مدارها
- اسکرین‌شات‌های شبیه‌سازی در Xilinx ISE
- پوشش کامل از سطح ترانزیستور تا پروژه‌های پیچیده

## محتوای آموزش

### بخش ۰۰: مقدمه و شروع کار
- Verilog چیست و چرا باید آن را یاد بگیریم؟
- نصب و راه‌اندازی Xilinx ISE
- ساخت اولین پروژه
- آشنایی با شبیه‌سازی

### بخش ۰۱: سطح ترانزیستور
- معرفی NMOS و PMOS
- مدل‌سازی Switch-Level
- ساخت گیت‌های منطقی با ترانزیستور
- CMOS Inverter و Transmission Gate

### بخش ۰۲: سطح گیت
- گیت‌های منطقی پایه
- Half Adder و Full Adder
- Ripple Adder
- Comparator و Multiplexer

### بخش ۰۳: سطح جریان داده
- انتساب پیوسته (Continuous Assignment)
- عملگرهای Verilog
- طراحی Adder، Decoder، Encoder
- Priority Encoder و Multiplexer

### بخش ۰۴: سطح رفتاری
- بلوک‌های Always و Initial
- Blocking و Non-blocking
- Latch و Flip-Flop
- Register و Counter
- شمارنده‌ها و رجیسترهای شیفت

### بخش ۰۵: موضوعات پیشرفته
- ماشین‌های حالت محدود (FSM)
- حافظه‌های RAM و ROM
- Functions و Tasks
- پروژه‌های کامل

## نحوه استفاده

۱. ابتدا بخش مقدمه را مطالعه کنید و Xilinx ISE را نصب کنید
۲. به ترتیب بخش‌ها را مطالعه کنید
۳. کدهای هر بخش را در ISE اجرا کنید
۴. تست‌بنچ‌ها را شبیه‌سازی کنید
۵. با تغییر پارامترها آزمایش کنید
