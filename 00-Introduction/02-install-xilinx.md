# نصب و راه‌اندازی Xilinx ISE

## دانلود

۱. به وب‌سایت Xilinx مراجعه کنید
۲. نسخه ISE Design Suite را دانلود کنید
۳. نسخه رایگان (WebPACK) برای شروع کافی است

## نصب در Windows

### مراحل نصب

۱. فایل نصب را اجرا کنید
۲. License رایگان را انتخاب کنید
۳. مسیر نصب را مشخص کنید
۴. منتظر تکمیل نصب بمانید

### تنظیمات اولیه

پس از نصب:

۱. نرم‌افزار ISE Design Suite را باز کنید
۲. از منوی File گزینه New Project را انتخاب کنید
۳. با محیط کاری آشنا شوید

## بخش‌های مهم محیط ISE

### Project Navigator
مدیریت فایل‌های پروژه

### Design Panel  
مشاهده فایل‌های طراحی

### Processes Panel
اجرای شبیه‌سازی و سنتز

### Console
نمایش پیام‌های خطا و هشدار

## آلترناتیو: Vivado

برای پروژه‌های جدیدتر می‌توانید از Vivado استفاده کنید که نسخه جدیدتر Xilinx است.

## تست نصب

برای اطمینان از نصب صحیح:

۱. یک پروژه جدید ایجاد کنید
۲. یک فایل Verilog اضافه کنید
۳. کد ساده‌ای بنویسید و Syntax Check کنید

اگر خطایی نداشت، نصب موفقیت‌آمیز بوده است.

## مشکلات رایج

### License Error
مطمئن شوید License رایگان را انتخاب کرده‌اید

### Path Error
از مسیرهایی که دارای کاراکتر فارسی هستند استفاده نکنید

### Missing Libraries
در صورت نیاز، کتابخانه‌های اضافی را نصب کنید

اکنون آماده ساخت اولین پروژه هستید!
